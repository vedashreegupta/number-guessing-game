# Guess the Number

A simple, browser-based number guessing game. The goal is to guess a randomly generated number between 1 and 100 in 10 attempts or less.

## How to Play

1. Open `index.html` in any web browser.
2. Enter a number between 1 and 100.
3. Click "submit guess" (or press Enter).
4. The page will display if your guess is too high or too low, and track your previous guesses.

## Key Features

- Game logic restricts play to exactly 10 attempts.
- Real-time high/low indicators.
- Displays a running list of previous guesses.
- Fully client-side (no server or build steps required).

## Project Structure

- `index.html` - Markup and form structure.
- `style.css` - Page styling and layout.
- `script.js` - Game logic, state management, and input validation.
- `.gitignore` - Standard files to ignore for Git version control.

## Technologies
